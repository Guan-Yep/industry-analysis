# 行业分析技能 - 使用说明

## 一、技能简介

本技能可根据用户提供的行业关键词，自动完成全流程行业分析，输出专业的行业分析报告和可视化PPT。

### 核心能力

| 能力 | 说明 |
|------|------|
| 🔍 智能搜索 | 自动扩充关键词，联网搜索20+篇行业资料 |
| 📊 多维分析 | PEST、BCG、SWOT等专业分析框架 |
| 📈 数据可视化 | Mermaid/ASCII图表自动生成 |
| 📑 报告生成 | 结构化Markdown报告 |
| 🎨 PPT制作 | 9页专业PPT图片 |

### 适用场景

- 行业研究与市场分析
- 投资决策与尽职调查
- 竞争环境分析
- 战略规划与汇报材料
- 会前准备与背景调研

## 二、快速开始

### 2.1 基础用法

只需告诉AI你想分析的行业：

```
请帮我分析新能源汽车行业
```

AI将自动完成：
1. 关键词扩充与联网搜索
2. 行业概览分析
3. PEST环境分析
4. BCG矩阵分析
5. SWOT战略分析
6. 生成完整报告（含图表）
7. 生成9页PPT

### 2.2 选择输出模式

| 模式 | 触发方式 | 输出内容 | 用时 |
|------|----------|----------|------|
| 速查模式 | "快速了解XX行业" | 1页速查卡 | 1分钟 |
| 完整模式 | "详细分析XX行业" | 完整报告+PPT | 10分钟 |
| 精简模式 | "简要分析XX行业" | 精简版报告 | 5分钟 |
| 汇报模式 | "XX行业汇报材料" | 执行摘要+核心洞察 | 3分钟 |
| 竞品模式 | "分析XX行业竞争格局" | 竞品对比报告 | 5分钟 |
| 趋势模式 | "XX行业趋势预测" | 趋势洞察+时机评估 | 5分钟 |

### 2.3 示例对话

**示例1：完整分析**

AI：好的，我将为您生成完整的人工智能行业分析报告...
```

**示例2：快速了解**

```

**示例3：竞品分析**

## 三、高级功能

### 3.1 数据增强（提升专业度）

如果您有行业报告或数据，可以上传让AI提取关键数据：

```
用户：我上传了一份行业报告，请提取关键数据后再分析
AI：好的，我将从您的报告中提取数据...
```

**支持提取的数据类型：**
- 市场规模、增长率、渗透率
- 市场份额、用户规模
- 营收、利润、投融资
- 企业数量、公司名称

**数据校验：** AI会自动检查数据合理性，标记异常值

**来源追踪：** 报告中会标注数据来源和可信度

### 3.2 报告定制化

**选择报告类型：**
```
用户：只需要PEST分析和SWOT分析
AI：好的，我将生成PEST+SWOT专题报告...
```

**选择输出风格：**
```
用户：用数据驱动的风格写报告
AI：好的，我将使用量化、精确的表达方式...
```

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| 正式风格 | 专业、严谨、客观 | 正式报告、外部汇报 |
| 简洁风格 | 直接、清晰、高效 | 内部沟通、快速决策 |
| 数据驱动 | 量化、精确、证据导向 | 投资分析、技术汇报 |

### 3.3 趋势洞察

**技术成熟度分析（Hype Cycle）：**
```
用户：分析新能源汽车相关技术的成熟度
AI：好的，我将生成Gartner技术成熟度曲线分析...
```

**进入时机评估：**
```
用户：现在进入新能源汽车行业合适吗？
AI：好的，我将从市场成熟度、竞争强度、技术成熟度、资源要求四个维度评估...
```

**风险预警：**
```
用户：新能源汽车行业有哪些风险？
AI：好的，我将生成风险预警清单，涵盖市场、技术、政策、竞争、运营五类风险...
```

## 五、常见问题

### Q1：分析需要多长时间？

| 模式 | 预计用时 |
|------|----------|
| 速查模式 | 1-2分钟 |
| 精简模式 | 3-5分钟 |
| 完整模式 | 8-12分钟 |
| 含PPT生成 | 额外3-5分钟 |

### Q2：可以分析哪些行业？

理论上可以分析任何行业，包括但不限于：
- 科技：人工智能、半导体、云计算、区块链
- 消费：新能源汽车、智能家居、宠物经济
- 医疗：创新药、医疗器械、数字医疗
- 金融：金融科技、保险科技
- 制造：智能制造、新材料、机器人

### Q3：数据来源是什么？

- 联网搜索的公开资料
- 用户上传的行业报告
- 官方统计数据
- 研究机构报告

报告中会标注数据来源和可信度。

### Q4：如何获得更准确的分析？

1. **提供更多上下文**：说明分析目的、关注重点
2. **上传行业报告**：提供权威数据源
3. **指定分析维度**：明确需要哪些分析模块
4. **反馈修正**：对初步结果提出修改意见

### Q5：图表显示异常怎么办？

如果Mermaid图表渲染不正常，可以要求：
```
用户：请用ASCII格式重新生成图表
AI：好的，我将使用ASCII格式生成兼容性更强的图表...
```

1. **明确目的**：说明是投资研究、竞争分析还是战略规划
