# 行业分析PPT生成模板

本文档提供使用GenerateImage工具生成行业分析PPT的详细指南。

## 视觉风格规范

### 尺寸规范

- **宽度**：2560px
- **高度**：1440px
- **比例**：16:9

### 整体风格

- **设计风格**：线性扁平风格
- **背景色调**：浅蓝灰色调，白色工程图纸感
- **纹理**：淡淡的工程图纸网格纹理
- **装饰**：极简几何线条装饰
- **图解元素**：少量扁平化图解元素
- **禁止元素**：不出现人像

### 字体规范

- **标题**：黑色粗体，字号小
- **正文**：深灰色，字号非常小
- **强调**：深蓝色
- **数据**：大号字突出显示

### 布局原则

- 留白充足
- 信息层次清晰
- 视觉重心明确
- 对齐规范统一

## PPT生成执行流程

### 步骤1：准备内容

在生成PPT之前，确保已完成：
- 行业概览分析
- PEST分析
- BCG矩阵分析
- SWOT分析
- 关键洞察提炼

### 步骤2：填充模板

根据分析结果，将具体内容填充到各页模板中。

### 步骤3：生成图片

使用GenerateImage工具，按顺序生成9页PPT图片：

```
GenerateImage调用示例：

description: [填充后的提示词]
filename: "ppt-page-1-cover.png"
```

## 内容精简原则

由于图片空间有限，内容需要精简：

1. **数据精简**：只展示最核心的3-5个数据点
2. **文字精简**：每个要点控制在15字以内
3. **层次精简**：每页最多3个层次
4. **结论明确**：每页底部一句话总结
