# 行业速查卡模板

本文档提供行业速查卡的模板和使用指南，用于快速了解一个行业的核心信息。

## 一、速查卡用途

### 适用场景

| 场景 | 说明 |
|------|------|
| 快速背调 | 拜访客户前5分钟了解其所在行业 |
| 会议准备 | 参会前快速掌握讨论行业的关键信息 |
| 投资初筛 | 快速判断一个赛道是否值得深入研究 |
| 谈资储备 | 积累行业知识，提升专业形象 |
| 汇报引言 | 作为详细报告的开篇概要 |

### 与完整报告的区别

| 维度 | 速查卡 | 完整报告 |
|------|--------|----------|
| 篇幅 | 1页 | 10-20页 |
| 深度 | 核心要点 | 详细分析 |
| 用时 | 1分钟阅读 | 30分钟阅读 |
| 目的 | 快速了解 | 深入研究 |
| 输出 | 数据+结论 | 分析+论证 |

---

## 二、速查卡内容结构

### 2.1 核心数据区（必填）

| 指标 | 说明 | 数据来源 |
|------|------|----------|
| 市场规模 | 中国市场/全球市场规模 | 行业报告、研究机构 |
| 增长率 | 同比增长率或CAGR | 统计数据、研究报告 |
| 发展阶段 | 萌芽期/成长期/成熟期/衰退期 | 综合判断 |
| 渗透率 | 产品/服务的市场渗透率 | 行业数据 |

### 2.2 竞争格局区（必填）

| 内容 | 说明 |
|------|------|
| 头部玩家 | TOP3-5企业及市场份额 |
| 格局特征 | 寡头/分散/一超多强等 |

### 2.3 驱动与风险区（必填）

| 内容 | 说明 |
|------|------|
| 核心驱动 | 推动行业增长的3个主要因素 |
| 主要风险 | 行业面临的3个主要风险 |

### 2.4 趋势判断区（必填）

| 内容 | 说明 |
|------|------|
| 整体判断 | 看多/看平/看空/谨慎 |
| 投资建议 | 一句话投资或关注建议 |

### 2.5 一句话总结（必填）

用一句话概括该行业的核心特征和发展态势。

## 三、数据收集指南

### 3.1 核心数据来源

| 数据类型 | 推荐来源 |
|----------|----------|
| 市场规模 | 艾瑞咨询、IDC、Gartner、各行业协会 |
| 增长率 | 国家统计局、行业年报、研究报告 |
| 企业份额 | 上市公司年报、行业排名、招股书 |
| 政策信息 | 政府官网、政策数据库 |

### 3.2 发展阶段判断标准

| 阶段 | 增长率 | 渗透率 | 竞争格局 | 典型特征 |
|------|--------|--------|----------|----------|
| 萌芽期 | >50% | <5% | 少数先驱 | 技术验证、商业模式探索 |
| 成长期 | 20-50% | 5-30% | 快速洗牌 | 资本涌入、玩家增多 |
| 成熟期 | 5-20% | 30-70% | 格局稳定 | 头部集中、效率竞争 |
| 衰退期 | <5% | >70% | 寡头垄断 | 增量转存量、整合并购 |

### 3.3 趋势判断标准

| 判断 | 依据 |
|------|------|
| 看多 📈 | 高增长+政策利好+技术突破+需求旺盛 |
| 看平 ➡️ | 稳定增长+格局稳定+无重大变量 |
| 看空 📉 | 增长放缓+政策收紧+替代品威胁 |
| 谨慎 ⚠️ | 存在重大不确定性，需持续观察 |

## 六、使用脚本

```bash
# 默认ASCII格式
python quick_card_generator.py "新能源汽车"

# 使用数据文件
python quick_card_generator.py "新能源汽车" data.json

# Markdown格式
python quick_card_generator.py "新能源汽车" data.json --format=markdown

# JSON格式
python quick_card_generator.py "新能源汽车" data.json --format=json
```
