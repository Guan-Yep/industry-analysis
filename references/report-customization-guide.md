# 报告定制化模块使用指南

本文档说明如何使用报告定制化模块，按需生成不同类型、风格的分析报告。

## 一、模块概览

| 功能 | 说明 |
|------|------|
| 报告裁剪器 | 按需选择分析模块，生成精简版/汇报版 |
| 风格转换器 | 转换报告语言风格（正式/简洁/数据驱动） |
| 摘要生成器 | 自动生成1页执行摘要 |
| 亮点提取器 | 提取3-5个核心洞察 |

## 二、报告类型

### 2.1 预设报告类型

| 类型代码 | 名称 | 包含模块 | 适用场景 |
|----------|------|----------|----------|
| `full` | 完整版 | 全部6个模块 | 深度研究、存档 |
| `brief` | 精简版 | 概览+SWOT+竞争 | 快速了解、日常参考 |
| `executive` | 汇报版 | 概览+SWOT | 高层汇报、决策参考 |
| `pest_only` | PEST专题 | 概览+PEST | 宏观环境分析 |
| `swot_only` | SWOT专题 | 概览+SWOT | 战略分析 |
| `competition_only` | 竞争专题 | 概览+竞争+BCG | 竞争格局分析 |

### 2.2 可选分析模块

| 模块代码 | 名称 | 包含内容 |
|----------|------|----------|
| `overview` | 行业概览 | 行业定义、市场规模、发展阶段、产业链 |
| `pest` | PEST分析 | 政治、经济、社会、技术环境 |
| `bcg` | BCG矩阵 | 明星、现金牛、问题、瘦狗业务 |
| `swot` | SWOT分析 | 优势、劣势、机会、威胁、战略建议 |
| `competition` | 竞争格局 | 市场份额、头部企业、竞争态势 |
| `trend` | 趋势展望 | 发展趋势、机会预测、风险预警 |

### 2.3 使用方法

```bash
# 查看所有配置选项
python report_customizer.py config

# 生成精简版报告
python report_customizer.py trim brief full_report.md

# 只保留PEST和SWOT模块
python report_customizer.py trim pest,swot full_report.md
```

## 三、输出风格

### 3.1 可用风格

| 风格代码 | 名称 | 特点 | 适用场景 |
|----------|------|------|----------|
| `formal` | 正式风格 | 专业、严谨、客观 | 正式报告、学术场景 |
| `concise` | 简洁风格 | 直接、清晰、高效 | 快速阅读、内部沟通 |
| `data_driven` | 数据驱动 | 量化、精确、证据导向 | 数据分析、投资决策 |

### 3.2 风格示例对比

**同一内容的不同风格表达：**

**正式风格**:
> 根据行业研究数据显示，2024年中国新能源汽车市场规模达到8000亿元，同比增长35%。从市场发展态势来看，这一增长趋势有望在未来三年内持续保持。

**简洁风格**:
> 2024年市场规模8000亿元，增长35%。未来3年将持续增长。

**数据驱动风格**:
> 数据显示：2024年市场规模8000亿元（+35% YoY），CAGR预计维持在25-30%。（数据来源：中汽协）

### 3.3 使用方法

```bash
# 获取风格写作指南
python report_customizer.py style formal
python report_customizer.py style concise
python report_customizer.py style data_driven
```

## 四、执行摘要

### 4.1 摘要结构

执行摘要是1页精华内容，包含：

| 模块 | 内容 |
|------|------|
| 核心发现 | 市场规模、增长率、发展阶段 |
| 竞争态势 | TOP3企业 |
| 战略要点 | 核心优势、主要风险、关键机会 |
| 行动建议 | 3条具体建议 |

### 4.2 使用方法

```bash
# 生成摘要模板
python report_customizer.py summary

# 使用数据生成摘要
python report_customizer.py summary data.json "新能源汽车"
```

### 4.3 摘要模板

```markdown
# 执行摘要模板

## 一句话总结
[用一句话概括行业现状和趋势]

## 三个核心数据
1. 市场规模：[X亿元]
2. 增长率：[X%]
3. 渗透率/集中度：[X%]

## 三个关键洞察
1. [洞察1]
2. [洞察2]
3. [洞察3]

## 三个行动建议
1. [建议1]
2. [建议2]
3. [建议3]

## 风险提示
[主要风险点]
```

## 五、核心洞察提取

### 5.1 洞察类型

| 类型 | 图标 | 关键词 |
|------|------|--------|
| 市场洞察 | 📈 | 市场规模、增长、份额、渗透率 |
| 竞争洞察 | 🏆 | 竞争、份额、头部、格局 |
| 技术洞察 | 🔬 | 技术、创新、研发、专利 |
| 风险洞察 | ⚠️ | 风险、挑战、威胁、不确定 |
| 机会洞察 | 🎯 | 机会、机遇、潜力、增长点 |

### 5.2 使用方法

```bash
# 获取洞察提取模板
python report_customizer.py insights

# 从报告中提取洞察
python report_customizer.py insights full_report.md
```

### 5.3 洞察输出格式

```markdown
### 洞察1: 市场高速增长
**类型**: 市场洞察
**内容**: 新能源汽车市场保持35%以上高速增长
**数据支撑**: 2024年市场规模8000亿元，同比+35%
**启示**: 市场仍处于快速扩张期，是进入的好时机
```

## 六、工作流程

### 6.1 定制化报告生成流程

```
确定报告用途和受众
       ↓
  选择报告类型（完整/精简/汇报）
       ↓
  选择分析模块（按需组合）
       ↓
  选择输出风格（正式/简洁/数据驱动）
       ↓
  生成定制化报告
       ↓
  生成执行摘要
       ↓
  提取核心洞察
```

### 6.2 场景化推荐

| 场景 | 推荐配置 |
|------|----------|
| 向CEO汇报 | 汇报版 + 正式风格 + 执行摘要 |
| 投资决策 | 完整版 + 数据驱动风格 |
| 团队内部分享 | 精简版 + 简洁风格 |
| 竞争分析会议 | 竞争专题 + 数据驱动风格 |
| 战略规划 | SWOT专题 + 正式风格 |

## 七、最佳实践

### 7.1 报告长度建议

| 报告类型 | 建议页数 | 阅读时间 |
|----------|----------|----------|
| 完整版 | 15-20页 | 30-45分钟 |
| 精简版 | 8-10页 | 15-20分钟 |
| 汇报版 | 3-5页 | 5-10分钟 |
| 执行摘要 | 1页 | 2-3分钟 |

### 7.2 风格选择建议

- **正式风格**: 外部报告、正式场合、存档文档
- **简洁风格**: 内部沟通、快速决策、日常参考
- **数据驱动**: 投资分析、量化评估、技术汇报
