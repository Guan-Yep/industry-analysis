# 行业分析框架详细指南

本文档提供PEST分析、BCG矩阵、SWOT分析的详细执行指南。

## 一、PEST分析框架

### 1.1 分析目的

PEST分析用于评估宏观环境对行业的影响，识别外部机会与威胁。

### 1.2 四维度详细指标

#### Political（政治法规）

**核心关注点**：仅检索政策法规相关内容

| 指标类别 | 具体内容 |
|----------|----------|
| 法律法规 | 行业相关法律、行政法规、部门规章 |
| 产业政策 | 国家/地方产业政策、发展规划、扶持政策 |
| 监管要求 | 准入门槛、资质要求、合规标准 |
| 行业标准 | 国家标准、行业标准、团体标准 |
| 税收政策 | 税收优惠、补贴政策 |
| 贸易政策 | 进出口政策、关税政策 |

**搜索关键词示例**：
- "[行业] 政策法规 2024"
- "[行业] 监管要求"
- "[行业] 国家标准"
- "[行业] 产业政策"

#### Economic（经济环境）

**核心关注点**：重点关注居民消费相关指标

| 指标类别 | 具体内容 |
|----------|----------|
| 宏观经济 | GDP增长率、通货膨胀率、利率水平 |
| 居民收入 | 人均可支配收入、收入增长率 |
| 消费能力 | 消费支出、消费结构、消费升级 |
| 行业投资 | 投融资规模、资本流向、估值水平 |
| 成本因素 | 原材料成本、人力成本、运营成本 |
| 汇率影响 | 汇率波动对进出口的影响 |

**搜索关键词示例**：
- "居民消费 [行业] 支出"
- "[行业] 市场规模 增长率"
- "[行业] 投融资 2024"
- "消费升级 [行业]"

#### Social（社会文化）

**核心关注点**：重点关注消费观念、消费习惯

| 指标类别 | 具体内容 |
|----------|----------|
| 人口结构 | 年龄结构、城镇化率、家庭结构 |
| 消费观念 | 消费理念变化、品质追求、性价比意识 |
| 消费习惯 | 购买渠道偏好、消费频次、决策因素 |
| 生活方式 | 工作生活平衡、健康意识、休闲方式 |
| 社会价值观 | 环保意识、社会责任、文化认同 |
| 教育水平 | 受教育程度、专业技能需求 |

**搜索关键词示例**：
- "[行业] 消费者行为"
- "[行业] 消费趋势"
- "Z世代 [行业] 消费"
- "[行业] 用户画像"

#### Technological（技术环境）

**核心关注点**：重点关注媒介、网络、内容产业相关技术创新

| 指标类别 | 具体内容 |
|----------|----------|
| 核心技术 | 关键技术突破、技术成熟度、技术壁垒 |
| 数字化 | 数字化转型、智能化应用、数据驱动 |
| 新兴技术 | AI应用、大数据、云计算、物联网 |
| 媒介技术 | 新媒体、社交平台、内容分发 |
| 网络基础 | 5G/6G、网络覆盖、带宽提升 |
| 研发投入 | 行业研发强度、专利数量、创新能力 |

**搜索关键词示例**：
- "[行业] 技术趋势 2024"
- "[行业] AI应用"
- "[行业] 数字化转型"
- "[行业] 技术创新"

## 二、BCG矩阵分析框架

### 2.1 分析目的

BCG矩阵用于分析业务组合或竞争格局，指导资源配置决策。

### 2.2 两个核心维度

| 维度 | 定义 | 衡量标准 |
|------|------|----------|
| 市场增长率 | 行业/细分市场的增长速度 | >10%为高增长，<10%为低增长 |
| 相对市场份额 | 相对于最大竞争对手的份额比 | >1为高份额，<1为低份额 |

### 2.3 四象限定义与策略

#### 明星业务 (Stars)
- **特征**：高增长、高份额
- **现金流**：大量投入，收益也高
- **策略**：加大投资，维持领先地位
- **典型表现**：市场领导者，快速成长

#### 现金牛业务 (Cash Cows)
- **特征**：低增长、高份额
- **现金流**：稳定现金流入
- **策略**：维持现状，收割利润
- **典型表现**：成熟市场的领导者

#### 问题业务 (Question Marks)
- **特征**：高增长、低份额
- **现金流**：需要大量投入
- **策略**：选择性投资或放弃
- **典型表现**：新进入者，潜力待验证

#### 瘦狗业务 (Dogs)
- **特征**：低增长、低份额
- **现金流**：微利或亏损
- **策略**：收缩或退出
- **典型表现**：衰退市场的弱势者

## 三、SWOT分析框架

### 3.1 分析目的

SWOT分析用于全面评估内外部环境，制定匹配的战略方案。

### 3.2 四维度定义

| 维度 | 类型 | 定义 | 关注点 |
|------|------|------|--------|
| Strengths | 内部/有利 | 组织的优势和能力 | 资源、能力、竞争优势 |
| Weaknesses | 内部/不利 | 组织的劣势和不足 | 短板、限制、改进空间 |
| Opportunities | 外部/有利 | 外部环境的机会 | 市场机会、政策利好、技术趋势 |
| Threats | 外部/不利 | 外部环境的威胁 | 竞争威胁、政策风险、技术替代 |

### 3.3 各维度分析要点

#### Strengths（优势）
- 技术优势：核心技术、专利、研发能力
- 资源优势：资金、人才、渠道、品牌
- 运营优势：效率、成本、质量、服务
- 市场优势：份额、客户关系、品牌认知

#### Weaknesses（劣势）
- 技术短板：技术差距、创新不足
- 资源限制：资金紧张、人才缺乏
- 运营问题：效率低下、成本过高
- 市场弱点：份额小、品牌弱、渠道窄

#### Opportunities（机会）
- 市场机会：需求增长、新市场、细分机会
- 政策机会：政策支持、监管放松
- 技术机会：技术突破、数字化转型
- 竞争机会：竞争格局变化、整合机会

#### Threats（威胁）
- 竞争威胁：新进入者、替代品、价格战
- 政策风险：监管收紧、政策变化
- 技术风险：技术替代、技术变革
- 市场风险：需求下降、经济波动

### 3.4 交叉策略分析

#### SO策略（进攻型）
- **逻辑**：利用优势抓住机会
- **目标**：最大化优势和机会
- **典型策略**：扩张、创新、领先

#### WO策略（改进型）
- **逻辑**：克服劣势抓住机会
- **目标**：利用机会弥补劣势
- **典型策略**：合作、学习、转型

#### ST策略（防御型）
- **逻辑**：利用优势应对威胁
- **目标**：用优势化解威胁
- **典型策略**：差异化、多元化、防守

#### WT策略（收缩型）
- **逻辑**：减少劣势避免威胁
- **目标**：最小化劣势和威胁
- **典型策略**：收缩、退出、止损

## 四、波特五力分析框架（补充）

### 4.1 五种竞争力量

1. **现有竞争者的竞争强度**
   - 竞争者数量和规模
   - 行业增长率
   - 产品差异化程度
   - 退出壁垒

2. **潜在进入者的威胁**
   - 进入壁垒高低
   - 规模经济效应
   - 品牌忠诚度
   - 资本需求

3. **替代品的威胁**
   - 替代品的可获得性
   - 替代品的性价比
   - 转换成本

4. **供应商的议价能力**
   - 供应商集中度
   - 供应商转换成本
   - 供应商前向整合能力

5. **购买者的议价能力**
   - 购买者集中度
   - 购买量占比
   - 产品差异化程度
   - 购买者后向整合能力

## 五、分析框架使用建议

### 5.1 分析顺序

1. **先外后内**：先用PEST分析宏观环境，再用SWOT分析内外结合
2. **先宏观后微观**：先看行业整体，再看具体企业
3. **先定性后定量**：先理解逻辑，再用数据支撑

### 5.2 数据来源优先级

1. 官方统计数据（国家统计局、行业协会）
2. 权威研究报告（艾瑞、IDC、Gartner等）
3. 上市公司年报、招股书
4. 行业媒体报道
5. 专家访谈、调研数据

### 5.3 常见分析误区

- **避免**：罗列信息而不分析
- **避免**：主观臆断缺乏数据支撑
- **避免**：各维度分析相互矛盾
- **避免**：结论与分析脱节
- **避免**：策略建议过于笼统

### 6.10 图表使用原则

#### 图表选择指南

| 数据类型 | 推荐图表 | 适用场景 |
|----------|----------|----------|
| 时间序列 | 折线图/柱状图 | 趋势展示、增长对比 |
| 占比分布 | 饼图 | 市场份额、成本结构 |
| 多维对比 | 象限图 | BCG矩阵、竞争定位 |
| 层级关系 | 思维导图 | PEST分析、知识结构 |
| 流程逻辑 | 流程图 | 产业链、商业模式 |
| 时间节点 | 时间线/甘特图 | 发展历程、项目规划 |

#### 图文并茂原则

1. **每个章节至少1张图表**：用可视化增强理解
2. **数据驱动**：图表必须有数据支撑，避免纯装饰
3. **图文呼应**：文字描述与图表内容相互印证
4. **简洁清晰**：避免信息过载，突出关键信息
5. **风格统一**：全文图表风格保持一致
